## Installation

JWWebImageCache is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'JWWebImageCache', :git => 'https://github.com/WJustin/JWWebImageCache.git'
```
